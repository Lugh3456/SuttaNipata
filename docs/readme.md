# Sutta Nipata

A bilingual study site for the Sutta Nipata (经集), one of the oldest texts in the Pali Canon. Spoke 8 under [The Dharma Gate](https://lugh3456.github.io/DharmaGate/) portal.

## Features

- 5 vaggas, 71 suttas
- Classical Chinese (T0198 义足经 for Vagga 4) and modern Chinese for all other vaggas
- Bhikkhu Bodhi English translation + commentary (toggle)
- Text-to-speech (🔊 聆听经文)
- Mobile-responsive

## Structure

```
index.html        — vagga-grouped sutta index
section1–71.html  — one page per sutta
css/style.css     — shared DharmaGate stylesheet
docs/             — project documentation
```
